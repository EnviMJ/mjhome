🗄️ 데이터베이스 (Database)

대규모 데이터를 체계적으로 저장, 관리, 조회하기 위한 시스템을 학습합니다.

Relational Model & SQL

  • ER 다이어그램을 통한 데이터 모델링
  • SQL 문법 (DDL, DML, DCL)
  • 정규화(Normalization) 과정
SQL 연습장

Transaction & Performance

  • ACID 특성 (원자성, 일관성, 고립성, 지속성)
  • 인덱스(Index)를 활용한 쿼리 최적화
  • NoSQL과 분산 데이터베이스의 이해
Key Point: 인덱스는 조회 속도를 높이지만 쓰기 속도는 늦출 수 있습니다.
