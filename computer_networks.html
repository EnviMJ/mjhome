+ 교과내용학 +

컴퓨터네트워크

+

OSI-7계층부터 최신 프로토콜 동향까지, 계층별 동작 원리를 흐름도와 함께 정리했습니다.

+
+ +
+
+

핵심 학습 영역

+
    +
  • OSI 7계층, TCP/IP 4계층 비교
    • +
  • 전송계층: TCP 3-way/4-way handshaking, 혼잡 제어
    • +
  • 네트워크 계층: IP 주소 체계, 라우팅, NAT, ICMP
    • +
  • 링크 계층: 이더넷, ARP, 스위칭/브리지, 오류 검출
    • +
+
+ +
+

대표 기출 포인트

+
    +
  • TCP/UDP의 차이와 활용 사례
    • +
  • IPv4 서브넷팅 계산, CIDR 마스크 변환
    • +
  • 라우팅 알고리즘(Bellman-Ford, Dijkstra) 비교
    • +
  • DNS, DHCP, HTTP/HTTPS 요청 흐름 추적
    • +
+
패킷 흐름을 요청 → 전달 → 응답 순서로 그려보며 각 계층의 역할을 표시하면 이해가 빠릅니다.
+
+ +
+

학습 팁 & 자료

+
    +
  • Wireshark 캡처 파일로 프로토콜 필드 직접 확인
    • +
  • 계층별 키워드(제어비트, 헤더 필드)를 카드로 암기
    • +
  • 기출 문제를 표 형태로 재정리해 암기 빈칸 만들기
    • +
