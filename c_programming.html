C프로그래밍

+

기초 문법부터 메모리 관리까지, 임용 1차에서 자주 출제되는 핵심 개념을 한눈에 볼 수 있도록 정리했습니다.

+
+ +
+
+

핵심 학습 영역

+
    +
  • 자료형, 포인터, 배열, 구조체, 열거형 등 기본 문법
    • +
  • 메모리 모델과 동적 할당 (malloc/free), 얕은/깊은 복사
    • +
  • 함수 포인터, 콜백, 파일 입출력
    • +
  • 전처리기, 매크로, 조건부 컴파일
    • +
+
+ +
+

대표 기출 포인트

+
    +
  • 포인터 산술과 다차원 배열 인덱싱
    • +
  • 메모리 누수, 이중 해제 등 오류 찾기
    • +
  • 문자열 처리와 버퍼 오버플로 취약점
    • +
  • 비트 연산을 활용한 플래그 관리
    • +
+
실제 코드를 손으로 작성하며 메모리 상태를 그림으로 표현하면 실수를 줄일 수 있습니다.

    • +
  • 토: 스스로 함수 단위로 리팩토링하며 주석으로 요약
